# Housing ML Fullstack Portal

A full-stack housing analytics platform that provides property price prediction and market analysis tools.

The system includes a machine learning model service for predicting housing prices and a web portal that allows users to estimate property values and explore market insights.

## Features

- Housing price prediction using a regression model
- REST API built with FastAPI
- Containerized ML service using Docker
- Multi-application portal built with Next.js
- Property value estimator interface
- Property market analytics dashboard

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend
- Python FastAPI
- Java Spring Boot

Machine Learning
- scikit-learn
- pandas

Infrastructure
- Docker
